# WinForm.UI
WinForm 皮肤，自定义控件
##### 使用方式：
###### BaseForm：
```C#
public partial class MainForm : BaseForm //修改父类 Form 为 BaseForm
```
###### Table ：
```C#
//默认提供两种适配器
private SimpleObjectAdapter<TradeBean> adapter;
private SimpleArrayAdapter adapter;
```
